FAQ

Common questions.

Can I switch plans?

Yes — upgrade or downgrade any time. Upgrades take effect immediately and we prorate. Downgrades apply at the next billing cycle. No phone calls, no penalties.

What does BYOK mean?

"Bring your own key." You provide your own API keys for Anthropic, OpenAI, and Google. PrivateForge orchestrates the agent and routes calls — but the calls go directly from your machine to the provider. We never proxy them, never see prompts, and never train on your data.

Where does my data live?

On the machine you installed PrivateForge on. Your code, your project memory, your history — all of it lives in your filesystem. Our cloud handles auth, billing, and the org directory; nothing else. The agent's actual work happens locally.

What happens after the 7-day trial?

If you don't add a payment method, the agent stops accepting new tasks but your local data and project memory stay on your machine — yours forever. Add a payment method any time to resume.

Do you offer a free tier?

No — but the 7-day free trial is the full product, every feature unlocked. We'd rather charge a fair flat fee than nickel-and-dime a "free" version into uselessness.

What about LLM costs?

You pay providers directly. Typical solo usage runs $20–$80/month in Anthropic or OpenAI charges; team usage scales with how aggressively the agent is run. Every call is itemized in PrivateForge so there are no surprises.

Can I run this on a server / VPC?

Yes. The agent is a standard binary — install it on your laptop, an EC2 box, a bare-metal server, anywhere you can run a Linux or macOS process. The web UI talks to your local agent over a localhost socket, or over your VPC's private network.
