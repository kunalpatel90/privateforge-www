PrivateForge — Spec sheet Today Version 1.0

What you actually get.

Every fact about the product, in a flat table. No animated charts, no hand-wavy adjectives.

№ 01 / Runtime

Where it runsmacOS 13+ · Windows 11. Native installer, no container.
Disk~280MB installed. Logs and memory grow with use.
MemoryIdle ~120MB. Per-task spikes vary by model.
NetworkOutbound only — to your provider APIs and connectors you authorize.
Offline modeLocal models work without internet. Cloud models require connectivity.

№ 02 / Privacy

Data routingDirect from your machine to the model provider. PrivateForge is not in the request path.
TrainingWe never train on your prompts, files, or memory. Provider terms apply for cloud calls.
LoggingLocal-only audit log of tool calls, file writes, and URLs visited. Replayable, exportable, deletable.
TelemetryOff by default. Opt-in, scoped to crash reports.
AccountRequired for cloud sync only. Most workflows run without one.
